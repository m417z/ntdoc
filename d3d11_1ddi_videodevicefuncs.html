D3D11_1DDI_VIDEODEVICEFUNCS structure

Description

The D3D11_1DDI_VIDEODEVICEFUNCS structure specifies the video function table for the Direct3D driver device object. This structure is used in Direct3D 11.1 drivers.

The driver must support the D3D 11.1 version of the user-mode DDI. When the DDI is supported, the runtime will call PFND3D10DDI_RETRIEVESUBOBJECT to retrieve the D3D11_1DDI_VIDEODEVICEFUNCS video device functions.
