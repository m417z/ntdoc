Windows Driver Kit DDI reference (ns-videoagp-_video_port_agp_services)

_VIDEO_PORT_AGP_SERVICES structure

Description

The VIDEO_PORT_AGP_SERVICES structure is obsolete and is supported only for backward compatibility with existing drivers. In its place, driver writers should use VIDEO_PORT_AGP_INTERFACE.

The VIDEO_PORT_AGP_SERVICES structure describes the AGP service routines provided by the video port driver. PnP video miniport drivers that can use AGP should call VideoPortGetAgpServices to initialize this structure. The video port driver initializes the entire structure; the miniport driver should never change any members.
