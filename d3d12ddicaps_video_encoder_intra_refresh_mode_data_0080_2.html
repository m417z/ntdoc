Windows Driver Kit DDI reference (ns-d3d12umddi-d3d12ddicaps_video_encoder_intra_refresh_mode_data_0080_2)

Description

The D3D12DDICAPS_VIDEO_ENCODER_INTRA_REFRESH_MODE_DATA_0080_2 structure is used to query whether a specified video encoding infra refresh mode is supported.

Members

NodeIndex

[in] In a multi-adapter operation, NodeIndex indicates which physical adapter of the device that the operation applies to.

Codec

[in] A D3D12DDI_VIDEO_ENCODER_CODEC_0080 value that specifies the codec to check support for.

Profile

[in] A D3D12DDI_VIDEO_ENCODER_PROFILE_DESC_0080_2 structure that specifies the profile to check support for.

Level

[in] A D3D12DDI_VIDEO_ENCODER_LEVEL_SETTING_0080_2 structure that specifies the level to check support for.

IntraRefreshMode

[in] A D3D12DDI_VIDEO_ENCODER_INTRA_REFRESH_MODE_0080 value that specifies the intra refresh mode to check support for.

IsSupported

[out] Indicates whether the given feature values are supported.

Remarks

The D3D runtime calls PFND3D12DDI_VIDEO_GETCAPS with D3D12DDICAPS_VIDEO_ENCODER_INTRA_REFRESH_MODE_DATA_0080_2 specified as the capability type.

See D3D12 video encoding for general information.
