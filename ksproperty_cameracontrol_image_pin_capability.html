Windows Driver Kit DDI reference (ne-ksmedia-ksproperty_cameracontrol_image_pin_capability)

KSPROPERTY_CAMERACONTROL_IMAGE_PIN_CAPABILITY enumeration

Description

Used to identify whether the camera's image pin and record pin are mutually exclusive. If they are mutually exclusive, then when the record pin is active, the image pin cannot be active, and vice-versa.

Constants

KSPROPERTY_CAMERACONTROL_IMAGE_PIN_CAPABILITY_PROPERTY_ID

The KSPROPERTY_CAMERACONTROL_IMAGE_PIN_CAPABILITY_PROPERTY property is used to identify whether the camera's image pin and the record pin are mutually exclusive.
