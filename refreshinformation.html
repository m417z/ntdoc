Windows hardware development documentation (refreshinformation)

RefreshInformation function

The RefreshInformation WMI method updates all tables for the HBA that corresponds to the calling instance object.

Parameters

This function has no parameters.

Return value

Not applicable.

Remarks

The RefreshInformation WMI method refreshes port attribute data that is retrieved by the following WMI methods:

GetDiscoveredPortAttributes

GetFC3MgmtInfo

GetFC4Statistics

GetFCPStatistics

GetPortAttributesByWWN

This WMI method belongs to the MSFC_HBAAdapterMethods WMI Class.
