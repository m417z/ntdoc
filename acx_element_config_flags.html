Windows Driver Kit DDI reference (ne-acxelements-acx_element_config_flags)

Description

The ACX_ELEMENT_CONFIG_FLAGS enumeration defines the config flags for an ACX element.

Constants

AcxElementConfigNoFlags

Indicates that no config flags are set.

AcxElementConfigValidFlags

For internal validation, do not use.

Remarks

There are no flags currently defined for this type.

ACX requirements

Minimum ACX version: 1.0

For more information about ACX versions, see ACX version overview.
