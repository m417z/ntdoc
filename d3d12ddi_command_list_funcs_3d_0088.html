Description

D3D12DDI_COMMAND_LIST_FUNCS_3D_0088 is used to provide pointers to the user-mode driver's command list 3D DDI functions for version Direct3D 12 Release 8, Build rev 8.

Remarks

The graphics kernel calls the user mode-driver's PFND3D12DDI_FILLDDITTABLE callback with a tableType value of D3D12DDI_TABLE_TYPE_COMMAND_LIST_3D to obtain the UMD's 3D command list functions for this DDI version.
