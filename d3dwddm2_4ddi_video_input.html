D3DWDDM2_4DDI_VIDEO_INPUT structure

Description

D3DWDDM2_4DDI_VIDEO_INPUT contains input information used by the PFND3D10DDI_RETRIEVESUBOBJECT DDI when querying the driver's WDDM 2.4 and later video functions.

Members

Relocate

Boolean value that indicates whether the functions can be relocated.

pWDDM2_4VideoDeviceFuncs

A pointer to a D3DWDDM2_4DDI_VIDEODEVICEFUNCS structure that contains pointers to the video functions being queried.
