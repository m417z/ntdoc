Windows Driver Kit DDI reference (nc-dot11wdi-miniport_wdi_rx_restart)

MINIPORT_WDI_RX_RESTART callback function

Description

[!IMPORTANT] This topic is part of the WDI driver model released in Windows 10. The WDI driver model is in maintenance mode and will only receive high priority fixes. WiFiCx is the Wi-Fi driver model released in Windows 11. We recommend that you use WiFiCx to take advantage of the latest features.

The MiniportWdiRxRestart handler function configures the RxEngine to restart indicating data traffic. This is issued following a MiniportWdiRxStop.

This is a WDI miniport handler inside NDIS_MINIPORT_WDI_DATA_HANDLERS.

Note You must declare the function by using the MINIPORT_WDI_RX_RESTART type. For more information, see the following Examples section.

Parameters

MiniportTalTxRxContext [in]

TAL device handle returned by the IHV miniport in MiniportWdiTalTxRxInitialize.

PortId [in]

The port ID.
