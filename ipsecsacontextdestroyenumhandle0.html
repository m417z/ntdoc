Windows Driver Kit DDI reference (nf-fwpmk-ipsecsacontextdestroyenumhandle0)

Description

The IPsecSaContextDestroyEnumHandle0 function frees a handle returned by IPsecSaContextCreateEnumHandle0.

Parameters

engineHandle [in]

Handle for an open session to the filter engine. Call FwpmEngineOpen0 to open a session to the filter engine.

enumHandle [in]

Handle of an IPsec security association (SA) context enumeration returned by IPsecSaContextCreateEnumHandle0.

Return value

Return code/value Description
ERROR_SUCCESS
0		 The enumerator was successfully deleted.
FWP_E_* error code
0x80320001—0x80320039		 A Windows Filtering Platform (WFP) specific error. See WFP Error Codes for details.
RPC_* error code
0x80010001—0x80010122		 Failure to communicate with the remote or local firewall engine.
Other NTSTATUS codes An error occurred.

Remarks

IPsecSaContextDestroyEnumHandle0 is a specific implementation of IPsecSaContextDestroyEnumHandle. See WFP Version-Independent Names and Targeting Specific Versions of Windows for more information.
