Windows Driver Kit DDI reference (ns-scsiscan-_scsiscan_info)

_SCSISCAN_INFO structure

Description

The SCSISCAN_INFO structure is used as a parameter to DeviceIoControl, when the specified I/O control code is IOCTL_SCSISCAN_GET_INFO.

Members

Size

Size, in bytes, of the SCSISCAN_INFO structure.

Flags

Not used, must be zero.

PortNumber

SCSI adapter number.

PathId

Host SCSI ID.

TargetId

Target SCSI ID.

Lun

Target logical unit number (LUN).

AdapterName

For internal use only.

Reserved

For internal use only.
