Windows Driver Kit DDI reference (nf-acxelements-acx_audioengine_config_init)

Description

The ACX_AUDIOENGINE_CONFIG_INIT function initializes a driver's ACX_AUDIOENGINE_CONFIG structure.

Parameters

cfg

An ACX_AUDIOENGINE_CONFIG structure.

Remarks

Example

Example usage is shown below.

  ACX_AUDIOENGINE_CONFIG          audioEngineCfg;

  ACX_AUDIOENGINE_CONFIG_INIT(&audioEngineCfg);

ACX requirements

Minimum ACX version: 1.0

For more information about ACX versions, see ACX version overview.
