Windows Driver Kit DDI reference (ne-ntifs-file_known_folder_type)

Description

A FILE_KNOWN_FOLDER_TYPE value identifies a known folder.

Constants

KnownFolderNone

The folder is not a known folder type.

KnownFolderDesktop

The Desktop folder.

KnownFolderDocuments

The Desktop folder.

KnownFolderDownloads

The Desktop folder.

KnownFolderMusic

The Desktop folder.

KnownFolderPictures

The Desktop folder.

KnownFolderVideos

The Desktop folder.

KnownFolderOther

The folder is a known folder type other than one of the above identified folders.

KnownFolderMax

The maximum value for this enumeration.
