Posts a work object to the thread pool. A worker thread calls the work object's callback function.

# Related Win32 API
[`SubmitThreadpoolWork`](https://learn.microsoft.com/en-us/windows/win32/api/threadpoolapiset/nf-threadpoolapiset-submitthreadpoolwork) maps directly to this function.
