_SRIOV_PF_EVENT enumeration

Description

Defines event values for the SR-IOV device.

Constants

SriovEventPfQueryStopDevice

The SR-IOV device is stopped.

SriovEventPfRestart

The SR-IOV device is restarted

SriovEventPfMaximum

Reserved.
